# Ahmed Hazem — Portfolio

This folder is ready to upload. Files included:
- index.html
- images/ (profile and project placeholders)
- cv.pdf (placeholder — replace with your real CV)

## How to use
1. Replace `images/profile.jpg` with your real photo (same file name).
2. Replace `cv.pdf` with your real CV file.
3. (Optional) Update project links in the Projects section.

## Deploy to GitHub Pages
- Create a repo (e.g., `portfolio`) and upload these files.
- In GitHub: Settings → Pages → Deploy from `main` branch → root.
- Your site will be available at: https://<your-username>.github.io/portfolio/

## Deploy to Netlify (drag & drop)
- Go to netlify.com → New site from Git.
- Or drag this folder into the Netlify UI.

Good luck!
